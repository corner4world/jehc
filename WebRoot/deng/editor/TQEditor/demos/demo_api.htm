编辑器对象方法:
appendHtml(string html) 最后的位置附加网页代码
changeMode(string mode) 切换可视化模式与代码模式, 空值时自动转另一种模式. mode可选值['code'|'visual']
cleanWord(bool force) 清理从word中粘贴的无用代码
content()获取代码
currentMode() /取得当前的编辑模式, code 代码 , visual 可视化
exeCmd(string cmd, string para) 执行特定的命令, 请参看W3C.
focus()使编辑器取得输入焦点
formatHTML(string html) 对代码进行美化处理
fullScreen(bool b)切换全屏与退出全屏
insertFlash(string url,[int width,[int height]]) 插入Flash
insertFlv(string url,[bool auto_start,[int width,[int height]]]) 插入Flv视频
insertHtml(string htm,[int|string position])在插入点插入HTML代码, position 为-1时, 有选中区前方插入, 1时在选中区后方插入, 0时替换选中区
insertImage(string image_url,[int width,[int height,[int border,[string alt,[string align]]]]]) 插入图片
insertMusic(string url,[bool auto_start]) 插入音频
insertRm(string url,[bool auto_start,[int width,[int height]]]) 插入RM视频
insertVideo(string url,[bool auto_start,[int width,[int height]]]) 插入一般视频
save() 保全代码至表单
setContent(string code)设置内容
setHeight(int value)设置高度,
setWidth(int|float value)设置宽度, 参数必须是0-1之间的小数表示百分比, 或大于1的整数表示像素值
typeset() 执行自动排版功能
